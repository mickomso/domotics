# DOMOTICS BACKEND *for Benimar Residential*

## *1. Introduction*
The project consists on a backend application development written in Java 11 with the Spring Boot framework.
It is going to be developed using Test Driven Development (TDD), trying to follow the Clean Code and SOLID principles.

## *2. Target*
The target is exposing a REST API uniquecly available from inside the local network installed in a flat, 
so that different devices and sensors can communicate each other.
